# 计划经济社群知识库
> 社群读书会全记录 | 核心文献翻译 | 经济模式百科



---

## 📌 仓库核心说明
本仓库是计划经济社群（PES）关于 **日常各项工作的文件备份和相关内容存储** 及民主经济模式的留存，**所有社群产出的翻译内容、读书会记录、模式百科资料，全部统一托管在此处**。该项目处于初期阶段，因此任然有相当多的不足之处。

---

## 📚 仓库包含的核心内容
### 1. 📝 社群读书会全记录
完整归档社群组织的每一期读书会资料，按场次、年份结构化整理，永久可追溯、可检索。
- 包含内容：每期读书会的预习材料、主讲内容纪要、社群讨论精华、答疑环节完整整理、回放与讲义附件
- 归档路径：`./读书会记录/20XX年X月/主题/文件名.xx` 文件夹，按「年份-主题」存放。

### 2. 📊 国际民主经济计划网络 经济模式百科
结构化梳理INDEP民主经济模式的开源百科，严谨、通俗地呈现完整理论与实践体系。
- 核心覆盖：核心概念权威释义、理论体系溯源、经济决策与分配框架、全球实践案例、INDEP组织动态、常见误区澄清
- 归档路径：`./国际民主经济计划网络/` 文件夹，按「百科-模式」分类存放

### 3. 📑 核心文献中文翻译库
托管社群完成校对的所有相关文献中文译本，版本可控、可追溯，接受社群校验与优化。
- 覆盖内容：罗宾·汉内尔、迈克尔·阿尔伯特等核心学者的经典著作/论文、INDEP官方纲领与研究报告、全球民主经济实践案例报告
- 归档路径：`./文献翻译/` 文件夹，按「著作、论文、官方文档、案例报告」分类存放
